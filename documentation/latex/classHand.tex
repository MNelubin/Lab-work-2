\doxysubsection{Detailed Description}
Represents a player\textquotesingle{}s hand containing various types of cards. 

The \doxylink{classHand}{Hand} class manages a collection of polymorphic \doxylink{classCard}{Card} objects, allowing for various operations like adding, removing, and sorting cards.
