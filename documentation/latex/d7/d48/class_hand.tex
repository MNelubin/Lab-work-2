\doxysubsection{Detailed Description}
Represents a player\textquotesingle{}s hand containing various types of cards. 

The \doxylink{class_hand}{Hand} class manages a collection of polymorphic \doxylink{class_card}{Card} objects, allowing for various operations like adding, removing, and sorting cards.

Generates a random card and adds it to the hand. 

This method randomly selects a card type and creates a card of that type with randomly generated properties. The card is then added to the hand.

The probability distribution for card types is as follows\+:
\begin{DoxyItemize}
\item Beast \doxylink{class_card}{Card}\+: 10\%
\item Buff \doxylink{class_card}{Card}\+: 10\%
\item Creature \doxylink{class_card}{Card}\+: 10\%
\item Shield \doxylink{class_card}{Card}\+: 10\%
\item Weapon \doxylink{class_card}{Card}\+: 10\%
\item Artifact \doxylink{class_card}{Card}\+: 10\%
\item Attack Spell \doxylink{class_card}{Card}\+: 10\%
