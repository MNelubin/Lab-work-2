Eats a card from the hand, increasing player\textquotesingle{}s HP based on card\textquotesingle{}s rarity and character\textquotesingle{}s HP multiplier. 

This method\+:
\begin{DoxyEnumerate}
\item Validates the card index
\item Gets the card\textquotesingle{}s rarity and converts it to a numerical value (Common=1, Uncommon=2, Rare=3, Epic=4)
\item Retrieves the character\textquotesingle{}s HP multiplier
\item Calculates the HP increase as (rarity value \texorpdfstring{$\ast$}{*} 10) multiplied by the HP multiplier
\item Converts the result to integer and adds it to player\textquotesingle{}s HP
\item Removes the card from the hand
\end{DoxyEnumerate}
