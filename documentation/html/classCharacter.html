Detailed Description

Represents a game character with progression system.

The Character class encapsulates all attributes and behaviors related to character progression, including experience points, leveling, and combat multipliers.

◆ lvl_up()

void Character::lvl_up ( )

Levels up the character.

Increases level by 1, reduces xp by xp_to_next_lvl, increases all multipliers by 0.05, and multiplies xp_to_next_lvl by 1.2. If xp >= new xp_to_next_lvl, recursively calls itself again.
