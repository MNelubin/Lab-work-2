Detailed Description

Represents a player's hand containing various types of cards.

The Hand class manages a collection of polymorphic Card objects, allowing for various operations like adding, removing, and sorting cards.

◆ generate_random_card()

void Hand::generate_random_card ( )

Generates a random card and adds it to the hand.

This method randomly selects a card type and creates a card of that type with randomly generated properties. The card is then added to the hand.

The probability distribution for card types is as follows:

  • Beast Card: 10%
  • Buff Card: 10%
  • Creature Card: 10%
  • Shield Card: 10%
  • Weapon Card: 10%
  • Artifact Card: 10%
  • Attack Spell Card: 10%
  • Heal Spell Card: 10%
  • Buff Spell Card: 10%
  • Defence Spell Card: 10%
